## 🚀 How to Run the Project Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/perashanid/To_do_for_bucc.git
   ```



2. Navigate to the project directory:

   ```bash
   cd To_do_for_bucc
   ```



3. Open the `index.html` file in your preferred web browser.
